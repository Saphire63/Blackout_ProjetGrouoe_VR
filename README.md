# Blackout — Projet VR Nyctophobie

## Assets à télécharger depuis l'Asset Store

Avant de lancer le projet, importer ces packages dans Unity :

- [ModularHousePack1](lien asset store)
- [Furniture Mega Pack](lien asset store)

## Setup

1. Cloner le repo
2. Ouvrir avec Unity 6 en URP
3. Importer les 3 assets ci-dessus
4. Ouvrir la scène `Scenes/jsp`
